## 🌀 Fractal Explorer

GPU-accelerated fractal rendering with WebGPU and WGSL shaders. Explore
Mandelbrot, Burning Ship, Tricorn, Phoenix, Julia sets, and more — with HDR
support, post-processing effects, and shareable URLs.

- 🔗 [WebGPU version](https://thaapasa.github.io/webgpu-fractal/examples/webgpu/)
  (Chrome 113+, Edge 113+)
- 🔗 [WebGL version](https://thaapasa.github.io/webgpu-fractal/examples/webgl/)
  (wider browser support)
- 📦 [Source code](https://github.com/thaapasa/webgpu-fractal)

## 🧱 Labyrinth Generator

Generates random labyrinths rendered at native device resolution for crisp,
sharp lines on any display. Uses a randomized wall-removal algorithm with
union-find and path compression to ensure every cell is reachable.

- 🔗 [Live demo](https://thaapasa.github.io/labyweb/)
- 📦 [Source code](https://github.com/thaapasa/labyweb)

## 🔬 Simulations

Interactive math and physics simulations: Langton's Ant, Conway's Game of Life,
Mandelbrot and Julia set explorers with progressive rendering and touch-friendly
pan/zoom. Built with React, TypeScript, and PixiJS.

- 🔗 [simulaatiot.pomeranssi.fi](https://simulaatiot.pomeranssi.fi/)
- 📦 [Source code](https://github.com/thaapasa/simulations)

## 🧮 Calculators

A collection of browser-based tools for time, numbers, colors, cryptography,
and data processing — with a chainable data pipeline supporting encoding, text
transforms, hashing, and compression. All calculations run locally in the
browser.

- 🔗 [laskurit.pomeranssi.fi](https://laskurit.pomeranssi.fi/)
- 📦 [Source code](https://github.com/thaapasa/calculators)
